# Node API Template

This starter is a small production-oriented Express baseline with:

- validated environment loading
- route registration separated from startup
- centralized JSON error handling
- request-level 404 fallback
- ESLint configuration
- Node test runner coverage for health and root routes

## Scripts

- `npm run dev`
- `npm start`
- `npm run lint`
- `npm test`

## Environment

Copy `.env.example` to `.env` and adjust values as needed.
