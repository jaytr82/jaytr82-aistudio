# Project Bootstrap Prompt

Bootstrap a new project from the requirements below using industry-standard structure, secure defaults, and testable design.

## Project Goal

`[Describe what the project should do.]`

## Constraints

- Language / framework: `[stack]`
- Runtime: `[node/python/go/etc.]`
- Database: `[postgres/mysql/sqlite/none]`
- Auth: `[jwt/session/oauth/none]`
- Deployment target: `[docker/vercel/aws/etc.]`

## Required Behavior

- Generate a sensible folder structure.
- Create the minimum core files needed to start development.
- Add configuration examples.
- Include validation and error-handling patterns.
- Recommend tests for critical paths.
- Prefer stable dependencies and simple architecture.

## Output Format

Provide:

1. file tree
2. complete starter files
3. setup instructions
4. next development milestones
